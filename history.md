## 需求

设计一个个人网站，用来帮助学习和个人总结。

## 设计方案

一个轻量的**个人知识中心**网站，围绕两个核心功能：

- **学习笔记** — 按科目组织，支持 Markdown，带标签系统
- **个人总结** — 用于写周记、月度回顾、学习反思，支持 Markdown 和标签

辅助功能：
- **仪表盘** — 统计概览、最近动态
- **科目管理** — 自定义学习领域，设定颜色标识
- **全站搜索** — 跨笔记和总结全文搜索

## 技术栈

| 层        | 选择                          |
|-----------|-------------------------------|
| 前端      | 纯 HTML / CSS / JavaScript    |
| 图标      | Lucide (CDN)                  |
| Markdown  | Marked.js (CDN)               |
| 字体      | Inter + Noto Sans SC (Google) |
| 数据存储  | 浏览器 localStorage           |

## 功能详情

### 仪表盘
- 统计卡片：笔记数、总结数、科目数、总数
- 最近更新的笔记列表（点击跳转详情）
- 最近的总结列表（点击跳转详情）

### 学习笔记
- 按科目筛选（科目筛选按钮组）
- 列表模式：标题、科目标签、更新时间、预览、标签、编辑/删除按钮
- 详情页：Markdown 渲染、返回列表、编辑、删除
- 新建/编辑表单：标题、所属科目、标签（回车添加/点击删除）、Markdown 内容

### 个人总结
- 列表模式：标题、更新时间、预览、标签、编辑/删除按钮
- 详情页：Markdown 渲染、返回列表、编辑、删除
- 新建/编辑表单：标题、标签、Markdown 内容

### 科目管理
- 网格展示：颜色圆点 + 科目名 + 笔记数
- 新增/编辑模态框：科目名 + 8色选色器
- 删除确认

### 搜索
- 顶部搜索框，输入 2 个字符后自动搜索
- 跨笔记标题/内容/标签 + 总结标题/内容/标签
- 分笔记和总结两组展示结果

### 交互细节
- 侧栏可折叠
- 模态弹窗新建/编辑
- 删除前确认
- Markdown 渲染：代码块、引用、列表、标题、图片等
- 响应式布局，移动端适配
- 全部数据存储在 localStorage

## 开发过程要点

1. 采用**工具风格**设计：安静的颜色、清晰的层级、适合阅读和扫描的排版
2. 遵循前端指引：不用装饰性卡片、渐变圆点，卡片圆角 ≤ 8px，用 Lucide 图标代替按钮文字
3. 全部数据存在 localStorage，开箱即用，零部署
4. JS 代码使用 IIFE 模式封装，无全局变量污染
5. CSS 变量体系方便后续主题定制
